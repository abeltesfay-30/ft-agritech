# FT Agritech Solutions Website

A modern, responsive startup website for FT Agritech Solutions - an Ethiopian AgriTech company developing smart IoT crop protection systems.

## 🌱 About

FT Agritech Solutions provides automated sensor-based crop protection systems with GSM SMS alerts, helping Ethiopian farmers reduce crop loss by 65-75% while freeing children from manual farm guarding duties.

**Founder:** Yordanos Abay  
**Program:** SEED Incubation Program  
**Location:** Ethiopia

## 🚀 Deployment

### Deploy to Vercel (Recommended)

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to project directory:
```bash
cd ft-agritech-website
```

3. Deploy:
```bash
vercel
```

### Deploy to Netlify

1. Drag and drop the entire folder to [Netlify Drop](https://app.netlify.com/drop)
2. Or use Netlify CLI:
```bash
npm install -g netlify-cli
netlify deploy
```

### Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Push your code:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
3. Enable GitHub Pages in repository settings

## ✨ Customization

### Update Contact Information

Edit the contact section in `index.html`:
```html
<p>ft.agritech@gmail.com</p>
<p>+251 XXX XXX XXX</p>
```

### Add Images

Place images in the `assets/images/` folder and update references in HTML:
```html
<img src="assets/images/your-image.jpg" alt="Description">
```

### Modify Colors

Update CSS variables in `style.css`:
```css
:root {
    --primary-green: #2ecc71;
    --tech-blue: #3498db;
    /* Add your custom colors */
}
```
