## 🚀 Quick Start (3 Steps)

### Step 1: View Your Website (30 seconds)

**Easiest way:**
1. Find `index.html` in this folder
2. Double-click it
3. Website opens in your browser!

**Better way (with auto-reload):**
1. Open folder in VS Code
2. Install "Live Server" extension
3. Right-click `index.html` → "Open with Live Server"

### Step 2: Customize (5 minutes)

**Update contact info in `index.html`:**
- Line ~580: Change email address

- Add product photos


2. Drag this folder

## 📧 Make Contact Form Work

The form has validation but needs a backend to send emails.

### Easiest: Formspree (2 minutes)

1. Go to [formspree.io](https://formspree.io)
2. Sign up free
3. Create form → Get form ID
4. Update `index.html` line ~605:

```html
<form action="https://formspree.io/f/YOUR-FORM-ID" method="POST">
```

### Alternative: Netlify Forms

If using Netlify, add to form tag:
```html
<form data-netlify="true" name="contact">
```
